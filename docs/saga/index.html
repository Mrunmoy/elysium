The Story of ms-os

Building a Real-Time Operating System from Scratch

This is the technical narrative of ms-os, a microkernel RTOS written in C++17 for ARM Cortex-M and Cortex-A processors. Each chapter covers one phase of development, from the first blinking LED to board-to-board communication over SPI and I2C.

Hardware Targets
Primary: STM32F207ZGT6 (Cortex-M3, 120 MHz)
Secondary: STM32F407ZGT6 (Cortex-M4, 168 MHz)
Tertiary: PYNQ-Z2 Zynq-7020 (Cortex-A9, 650 MHz)
Language: C++17 + Assembly
Tests: 516 C++ + 145 Python
Architecture: Microkernel

Debugging Playbook

JTAG setup, GDB recipes, crash dumps, real bug hunts, and toolchain tips.

Architecture Guide

Memory maps, boot sequence, threads, context switch, IPC, and register reference.
